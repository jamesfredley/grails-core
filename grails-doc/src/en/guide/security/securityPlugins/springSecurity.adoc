The Spring Security plugins are built on the https://spring.io/projects/spring-security[Spring Security] project which provides a flexible, extensible framework for building all sorts of authentication and authorization schemes. The plugins are modular so you can install just the functionality that you need for your application. The Spring Security plugins are the official security plugins for Grails and are actively maintained and supported.

There is a https://github.com/apache/grails-spring-security[Spring Security Core] plugin which supports form-based authentication, encrypted/salted passwords, HTTP Basic authentication, etc. and secondary dependent plugins provide alternate functionality such as https://github.com/apache/grails-spring-security[ACL support], https://github.com/apache/grails-spring-security[single sign-on with Jasig CAS], https://github.com/apache/grails-spring-security[LDAP authentication], https://github.com/grails-plugins/grails-spring-security-kerberos[Kerberos authentication], and a plugin providing https://github.com/apache/grails-spring-security[user interface extensions] and security workflows.

See the https://github.com/apache/grails-spring-security[Spring Security Core] plugin page for basic information and https://apache.github.io/grails-spring-security/latest/index.html[user guide] for detailed information.
